## About

This is my personal portfolio website — a single-page site showcasing my work in cybersecurity, including internship experience, security projects, and technical skills. I built it as a static site so it loads fast, stays simple, and runs free on GitHub Pages.

The design leans into a dark, terminal-inspired aesthetic that reflects my interest in security operations and systems work.

---

## What's inside

- **About** — who I am and what I focus on
- **Experience** — internships at TLT and Upskills Campus
- **Projects** — reverse proxy server, governance & risk assessment, federal court database with RBAC, and an agent-based traffic simulation
- **Skills** — languages, security tools, cloud platforms, and frameworks
- **Education & Certifications** — UNT cybersecurity degree, ISC2 CC and Security+ in progress
- **Contact** — email, LinkedIn, GitHub

---

## Tech stack

- **HTML5** & **CSS3** — single-file, no framework
- **Vanilla JavaScript** — scroll reveal animations via IntersectionObserver
- **Google Fonts** — Fraunces (serif) & JetBrains Mono
- **GitHub Pages** — hosting

No build step, no dependencies, no npm install. Just open `index.html` and it works.

---

## Running locally

```bash
git clone https://github.com/YOUR-USERNAME/YOUR-USERNAME.github.io.git
cd YOUR-USERNAME.github.io
open index.html
```

That's it. The site is one HTML file with all CSS and JS inline.
